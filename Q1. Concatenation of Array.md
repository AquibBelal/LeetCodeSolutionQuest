# 1929 - Concatenation of Array

## Intuition

We need to create a new array that contains the original array twice in sequence.

Two possible approaches:

1. Iterate through the input array twice and append each element to the result.
2. Create the result array with size `2 * n` beforehand and, during a single traversal, place each element at positions `i` and `i + n`.

The second approach is more efficient because it only requires one pass through the input array.

## Approach

1. Calculate the length of the input array `n`.
2. Create an output array `ans` of size `2 * n`.
3. Traverse the input array once.
4. For each index `i`:
   - Store `nums[i]` at `ans[i]`.
   - Store `nums[i]` again at `ans[i + n]`.
5. Return `ans`.

## Complexity Analysis

- **Time Complexity:** `O(n)`
- **Space Complexity:** `O(n)`

## Solution

```python
class Solution(object):
    def getConcatenation(self, nums):
        """
        :type nums: List[int]
        :rtype: List[int]
        """
        n = len(nums)
        ans = [0] * (2 * n)

        for i in range(n):
            ans[i] = nums[i]
            ans[i + n] = nums[i]

        return ans
```
