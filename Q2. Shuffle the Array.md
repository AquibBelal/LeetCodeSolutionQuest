# 1470. Shuffle the Array

## Intuition

The array is divided into two halves:

- First half: `[x1, x2, ..., xn]`
- Second half: `[y1, y2, ..., yn]`

We need to create a new array by taking one element from the first half and then one element from the second half repeatedly:

`[x1, y1, x2, y2, ..., xn, yn]`

To achieve this, we can traverse the first half of the array and, for each index `i`, append both `nums[i]` and `nums[i + n]` to the result array.

## Approach

1. Create an empty result array `ans`.
2. Iterate from `0` to `n - 1`.
3. For each index `i`:
   - Append `nums[i]` (the current `x` element).
   - Append `nums[i + n]` (the corresponding `y` element).
4. Return the resulting array.

## Complexity Analysis

- **Time Complexity:** `O(n)`
  - We iterate through the first half of the array once.
  - Each iteration performs two append operations, both of which are `O(1)`.

- **Space Complexity:** `O(n)`
  - The output array stores `2n` elements.
  - Ignoring constant factors, this requires `O(n)` extra space.

## Solution

```python
class Solution(object):
    def shuffle(self, nums, n):
        """
        :type nums: List[int]
        :type n: int
        :rtype: List[int]
        """
        ans = []

        for i in range(n):
            ans.append(nums[i])
            ans.append(nums[i + n])

        return ans
```
