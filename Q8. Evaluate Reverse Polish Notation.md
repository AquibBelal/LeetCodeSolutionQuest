# 150. Evaluate Reverse Polish Notation

## Intuition

Reverse Polish Notation (RPN) places operators after their operands.

For example:

```text
["2", "1", "+", "3", "*"]
```

represents:

```text
(2 + 1) * 3
```

A stack is the ideal data structure for this problem because:

- Numbers are pushed onto the stack.
- When an operator is encountered, the two most recent numbers are popped from the stack.
- The operation is performed and the result is pushed back onto the stack.
- At the end, the stack contains the final answer.

## Approach

1. Create an empty stack.
2. Traverse each token in the input array.
3. If the token is a number:
   - Push it onto the stack.
4. If the token is an operator (`+`, `-`, `*`, `/`):
   - Pop the top two numbers from the stack.
   - Apply the operation in the correct order.
   - Push the result back onto the stack.
5. After processing all tokens, the stack contains a single element, which is the answer.
6. Return the top element of the stack.

### Important Note for Division

The problem requires division to truncate toward zero.

In Python:

```python
int(a / b)
```

truncates toward zero and satisfies the problem requirements.

Examples:

```python
int(6 / -132)  # 0
int(-7 / 2)    # -3
```

## Complexity Analysis

- **Time Complexity:** `O(n)`
  - Each token is processed exactly once.

- **Space Complexity:** `O(n)`
  - In the worst case, all numbers may be stored in the stack.

## Solution

```python
class Solution(object):
    def evalRPN(self, tokens):
        """
        :type tokens: List[str]
        :rtype: int
        """
        stack = []

        for token in tokens:
            if token in ["+", "-", "*", "/"]:
                second = stack.pop()
                first = stack.pop()

                if token == "+":
                    stack.append(first + second)

                elif token == "-":
                    stack.append(first - second)

                elif token == "*":
                    stack.append(first * second)

                else:
                    stack.append(int(first / second))

            else:
                stack.append(int(token))

        return stack[-1]
```
