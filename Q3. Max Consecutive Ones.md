# 485. Max Consecutive Ones

## Intuition

We need to find the maximum number of consecutive `1`s in the array.

While traversing the array:

- Keep a counter to track the current streak of consecutive `1`s.
- If the current element is `1`, increment the counter.
- If the current element is `0`, compare the current streak with the maximum streak found so far and update it if necessary. Then reset the counter.
- An important edge case occurs when the array ends with consecutive `1`s. In that case, the final streak would not be compared with the maximum streak inside the loop, so we perform one last check before returning the answer.

## Approach

1. Initialize two variables:
   - `maxOnes` to store the maximum consecutive `1`s found so far.
   - `counter` to store the current streak of consecutive `1`s.
2. Traverse the array.
3. If the current element is `1`, increment `counter`.
4. If the current element is `0`:
   - Update `maxOnes` if `counter` is greater.
   - Reset `counter` to `0`.
5. After the traversal, check if the final streak stored in `counter` is greater than `maxOnes`.
6. Return the maximum consecutive `1`s.

## Complexity Analysis

- **Time Complexity:** `O(n)`
  - We traverse the array exactly once.

- **Space Complexity:** `O(1)`
  - Only two variables are used regardless of the input size.

## Solution 1

```python
class Solution(object):
    def findMaxConsecutiveOnes(self, nums):
        """
        :type nums: List[int]
        :rtype: int
        """
        maxOnes = 0
        counter = 0

        for i in range(len(nums)):
            if nums[i] == 1:
                counter += 1
            else:
                if counter > maxOnes:
                    maxOnes = counter
                counter = 0

        if counter > maxOnes:
            return counter

        return maxOnes
```

## Solution 2

```python
class Solution(object):
    def findMaxConsecutiveOnes(self, nums):
        """
        :type nums: List[int]
        :rtype: int
        """
        count = 0
        max_count = 0

        for num in nums:
            if num == 1:
                count += 1
            else:
                max_count = max(max_count, count)
                count = 0

        max_count = max(max_count, count)

        return max_count
```
