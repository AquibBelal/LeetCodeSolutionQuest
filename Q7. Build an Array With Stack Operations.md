# 1441. Build an Array With Stack Operations

## Intuition

We need to construct the `target` array using numbers from `1` to `n`.

For each number:

- If the number is part of the target array, we perform a `"Push"` operation.
- If the number is not part of the target array, we perform a `"Push"` followed immediately by a `"Pop"` to remove it.
- Once we have built the entire target array, we can stop processing further numbers.

## Approach

1. Initialize an empty list `output` to store the operations.
2. Initialize another list `duplicate_target` to track the elements that have been successfully built.
3. Iterate through numbers from `1` to `n`.
4. For each number:
   - If it exists in `target`:
     - Add `"Push"` to the operations list.
     - Add the number to `duplicate_target`.
   - Otherwise:
     - Add `"Push"` and `"Pop"` to the operations list.
5. After each iteration, check if `duplicate_target` matches `target`.
6. If they match, stop the process and return the operations performed.

## Complexity Analysis

- **Time Complexity:** `O(n × m)`
  - The loop runs up to `n` times.
  - The operation `i in target` takes `O(m)` time, where `m` is the length of `target`.
  - Therefore, the overall complexity is `O(n × m)`.

- **Space Complexity:** `O(m)`
  - `duplicate_target` stores up to `m` elements.
  - The output list stores the required operations.
  - Excluding the returned result, the auxiliary space is `O(m)`.

## Solution

```python
class Solution(object):
    def buildArray(self, target, n):
        """
        :type target: List[int]
        :type n: int
        :rtype: List[str]
        """
        output = []
        duplicate_target = []

        for i in range(1, n + 1):
            if i in target:
                output.append("Push")
                duplicate_target.append(i)
            else:
                output.append("Push")
                output.append("Pop")

            if duplicate_target == target:
                break

        return output
```

## Optimal solution

## Intuition

The numbers are read sequentially from `1` to `n`, and the target array is strictly increasing.

Instead of checking whether each number exists in `target`, we can use a pointer to track the current element we are trying to build.

For every number:

- If it matches the current target element, perform a `"Push"` and move the pointer forward.
- Otherwise, perform a `"Push"` followed by a `"Pop"` because that number is not needed.

Once all target elements have been constructed, we can stop immediately.

## Approach

1. Initialize an empty list `operations`.
2. Initialize a pointer `target_index = 0`.
3. Iterate through numbers from `1` to `n`.
4. If the current number matches `target[target_index]`:
   - Add `"Push"` to the operations list.
   - Move the pointer to the next target element.
5. Otherwise:
   - Add `"Push"` and `"Pop"` to the operations list.
6. If all target elements have been processed, stop the iteration.
7. Return the operations list.

## Complexity Analysis

- **Time Complexity:** `O(target[-1])`
  - We iterate only until the last element of the target array is reached.
  - Each number is processed exactly once.

- **Space Complexity:** `O(1)`
  - Only a pointer variable is used.
  - The returned operations list is not counted as auxiliary space.

## Solution

```python
class Solution(object):
    def buildArray(self, target, n):
        """
        :type target: List[int]
        :type n: int
        :rtype: List[str]
        """
        operations = []
        target_index = 0

        for num in range(1, n + 1):
            if num == target[target_index]:
                operations.append("Push")
                target_index += 1

                if target_index == len(target):
                    break
            else:
                operations.append("Push")
                operations.append("Pop")

        return operations
```
